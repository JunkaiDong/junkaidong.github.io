Golden-Thompson Inequality.

Statement. For all Hermitian n×nn\times nn×n matrices AAA and BBB,

Tr[eAeB]≥Tr[eA+B]\mathrm{Tr}[e^A e^B]\geq \mathrm{Tr}[e^{A+B}]Tr[eAeB]≥Tr[eA+B]

.

Proof. By Trotterization, we write eA+B=lim⁡N→∞(eA/NeB/N)Ne^{A+B} = \lim_{N\to\infty} (e^{A/N}e^{B/N})^NeA+B=limN→∞​(eA/NeB/N)N. Without loss of generality we can take N=2MN=2^MN=2M. We now only need to prove that for any matrix XXX and YYY,

Tr[(XY)2M]≤Tr[X2MY2M],\mathrm{Tr}[(XY)^{2^M}]\leq \mathrm{Tr}[X^{2^M}Y^{2^M}],Tr[(XY)2M]≤Tr[X2MY2M],

in which if we take X=eA/2M,Y=eB/2MX=e^{A/2^M}, Y=e^{B/2^M}X=eA/2M,Y=eB/2M we obtain the Golden-Thompson inequality.

We now use the corollary above:

Tr[(XY)2M]=Tr[((XY)2)2M−1]≤Tr[(XYY†X†)2M−1]=Tr[(X2Y2)2M−1].\mathrm{Tr}[(XY)^{2^M}] =\mathrm{Tr}[((XY)^2)^{2^{M-1}}] \leq \mathrm{Tr}[(XYY^\dagger X^\dagger)^{2^{M-1}}] = \mathrm{Tr}[(X^2Y^2)^{2^{M-1}}].Tr[(XY)2M]=Tr[((XY)2)2M−1]≤Tr[(XYY†X†)2M−1]=Tr[(X2Y2)2M−1].

Now, we can also repeat this procedure:

Tr[(X2Y2)2M−1]=Tr[((X2Y2)2)2M−2]≤Tr[(X4Y4)2M−2].\mathrm{Tr}[(X^2Y^2)^{2^{M-1}}] = \mathrm{Tr}[((X^2Y^2)^2)^{2^{M-2}}]\leq \mathrm{Tr}[(X^4Y^4)^{2^{M-2}}].Tr[(X2Y2)2M−1]=Tr[((X2Y2)2)2M−2]≤Tr[(X4Y4)2M−2].

Repeating this procedure for MMM times yields the inequality above.

Spin State of Halperin (mmn)(mmn)(mmn) States

This note aims to clarify many folklores that I have heard for a long time: that the Halperin (112)(112)(112) state is a singlet; that the (n+1,n+1,n)(n+1,n+1,n)(n+1,n+1,n) state that is a singlet; that all other (mmn)(mmn)(mmn) states are not singlets. I thank Patrick J. Ledwith and Tomohiro Soejima for providing some of the crucial arguments discussed here. Main references are David Tong's Notes on quantum Hall effects and Jainendra Jain's book Composite Fermions.

The Halperin (mmn)(mmn)(mmn) states

The Halperin (mmn)(mmn)(mmn) state is defined as

Ψ({zi},{wi})=∏i<jNz(zi−zj)m∏i<jNw(wi−wj)m∏i,j(zi−wj)e−ϕ({zi},{wi}),\Psi(\{z_i\},\{w_i\}) = \prod_{i \left.<\right. j}^{N_z} (z_i-z_j)^m \prod_{i \left.<\right. j}^{N_w} (w_i-w_j)^m \prod_{i,j}(z_i-w_j)e^{-\phi(\{z_i\},\{w_i\})},Ψ({zi​},{wi​})=i<j∏Nz​​(zi​−zj​)mi<j∏Nw​​(wi​−wj​)mi,j∏​(zi​−wj​)e−ϕ({zi​},{wi​}),

in which ϕ({zi},{wi})=−(∑i∣zi∣2−∑i∣wi∣2)/4ℓ2\phi(\{z_i\},\{w_i\}) = -(\sum_i |z_i|^2-\sum_i |w_i|^2)/4\ell^2ϕ({zi​},{wi​})=−(∑i​∣zi​∣2−∑i​∣wi​∣2)/4ℓ2 describes the Gaussian factor of the lowest Landau level. In practice, this state should be interpreted as a two-component many-body state:

∣Ψ⟩=∫DziDwiΨ({zi},{wi})∏ic↑†(zi)∏ic↓†(wi)∣0⟩.\ket{\Psi} = \int Dz_i Dw_i\Psi(\{z_i\},\{w_i\}) \prod_i c^\dagger_{\uparrow}(z_i) \prod_i c^\dagger_{\downarrow}(w_i) \ket{0}.∣Ψ⟩=∫Dzi​Dwi​Ψ({zi​},{wi​})i∏​c↑†​(zi​)i∏​c↓†​(wi​)∣0⟩.

∣Ψ⟩\ket{\Psi}∣Ψ⟩ is an eigenstate of SzS_zSz​ with eigenvalue (N↑−N↓)/2(N_\uparrow-N_\downarrow)/2(N↑​−N↓​)/2. It describes the Abelian Chern-Simons theory with

K=(mnnm),t=(1,1).K = \begin{pmatrix} m & n \\ n & m \end{pmatrix}, \quad t = (1,1).K=(mn​nm​),t=(1,1).

The ground state degeneracy on a torus is ∣det⁡K∣=∣m2−n2∣|\det K| = |m^2-n^2|∣detK∣=∣m2−n2∣. (Note that KKK is assumed to be non-singular; the (mmm)(mmm)(mmm) state describes a Laughlin ν=1/m\nu=1/mν=1/m state that has degeneracy mmm.) The filling at which this state can occur is

ν=tTK−1t=2m−2nm2−n2=2m+n \nu = t^T K^{-1} t = \frac{2m-2n}{m^2-n^2}=\frac{2}{m+n} ν=tTK−1t=m2−n22m−2n​=m+n2​

Due to Galilean symmetry, σxy=ν\sigma_{xy} = \nuσxy​=ν.

These are the basic features of the (mmn)(mmn)(mmn) state. We will now set Sz=0S_z=0Sz​=0 and seek the answer to the question: which of these (mmn)(mmn)(mmn) states are singlets? To do this we need an interesting condition called the Fock's cyclic condition (FCC).

Fock's cyclic condition

The FCC states the following condition. Given a many-body state Ψ({zi},{wj})\Psi(\{z_i\},\{w_j\})Ψ({zi​},{wj​}), in which ziz_izi​ describes the positions of the up-spin electrons and wjw_jwj​ describes the positions of the down-spin electrons, we say the state satisfies the FCC condition if and only if

∑i(zi,wj)Ψ=Ψ \sum_{i} (z_i, w_j)\Psi = \Psi i∑​(zi​,wj​)Ψ=Ψ

where (zi,wj)(z_i, w_j)(zi​,wj​) means that one switches the position of ziz_izi​ and wjw_jwj​ in the wavefunction. Although this might seem quite bizarre, it implies something deep: one can prove that, Ψ\PsiΨ has total angular momentum S=SzS=S_zS=Sz​ if and only if FCC holds; in other words, S+Ψ=0S^+ \Psi = 0S+Ψ=0.

To prove this think about the action of S+S^+S+ on Ψ\PsiΨ; this is going to split into S+=∑jSj+S^+ = \sum_j S_j^+S+=∑j​Sj+​. We will now show that S1+Ψ=0S_1^+\Psi=0S1+​Ψ=0 is equivalent to the FCC condition with j=1j=1j=1; one can convince themselves that given fermionic exchange, the FCC statement shouldn't depend on jjj.

With the many-body state (2) we write down the action of S1+S_1^+S1+​:

S1+∣Ψ⟩=∫DziDwjΨ({zi},{wj})∏ic↑†(zi)c↑†(w1)∏j≠1c↓†(wj)∣0⟩ S_1^+\ket{\Psi} = \int Dz_i Dw_j \Psi(\{z_i\},\{w_j\})\prod_i c^\dagger_{\uparrow}(z_i)c^\dagger_{\uparrow}(w_1)\prod_{j\neq 1} c^\dagger_{\downarrow}(w_j) \ket{0} S1+​∣Ψ⟩=∫Dzi​Dwj​Ψ({zi​},{wj​})i∏​c↑†​(zi​)c↑†​(w1​)j=1∏​c↓†​(wj​)∣0⟩

Now if this state is zero, all the wavefunction amplitudes must be zero. What is the wavefunction amplitude corresponding to the component ∏ic↑†(zi)c↑†(w1)∏j≠1c↓†(wj)∣0⟩\prod_i c^\dagger_{\uparrow}(z_i)c^\dagger_{\uparrow}(w_1)\prod_{j\neq 1} c^\dagger_{\downarrow}(w_j) \ket{0}∏i​c↑†​(zi​)c↑†​(w1​)∏j=1​c↓†​(wj​)∣0⟩? It is simply Ψ({zi},{wj})\Psi(\{z_i\},\{w_j\})Ψ({zi​},{wj​}), anti-symmetrized over all the up-spin electron coordinates (the down-spins have already been anti-symmetrized.) It is then not hard to see that the antisymmetrization gives the wavefunction amplitude of Ψ−∑i(zi,w1)Ψ\Psi - \sum_i (z_i, w_1)\PsiΨ−∑i​(zi​,w1​)Ψ.

Which are singlets?

We now have a powerful tool to check whether the state could be a singlet. Given that for a state to be a singlet, N↑N_\uparrowN↑​ must be equal to N↓N_\downarrowN↓​, or SzS_zSz​ must be zero, it is a singlet if and only if it satisfies the FCC.

We will now prove the only singlets are of type (n+1,n+1,n)(n+1,n+1,n)(n+1,n+1,n). To see that (n+1,n+1,n)(n+1,n+1,n)(n+1,n+1,n) is a singlet, note that it can be seen as symmetrically attaching nnn vortices to a fully filled (with both spins) LLL wavefunction. Given that symmetrically attaching vortices don't change the SU(2)SU(2)SU(2) symmetry (check for yourself!), and that a fully filled band is a singlet, the state must be a singlet. Proving that all others are not singlets is much more tricky.

Preemptively, we will call our chosen wjw_jwj​ to be risen z0z_0z0​. We will use the sign convention that the wavefunction is written as

Ψ=∏i<j(zi−zj)m(wi−wj)m∏i,j(zj−wi)n∏i(z0−wi)m(z0−zi)n \Psi = \prod_{i\left.<\right. j} (z_i-z_j)^m(w_i-w_j)^m \prod_{i, j} (z_j-w_i)^n \prod_i (z_0-w_i)^m (z_0-z_i)^n Ψ=i<j∏​(zi​−zj​)m(wi​−wj​)mi,j∏​(zj​−wi​)ni∏​(z0​−wi​)m(z0​−zi​)n

where the Gaussian factor has been premptively omitted. We will use the convention that Latin indices i,j,…i,j,\dotsi,j,… will not run through 000.

This case is trivial; as long as the particle number is large, FFF is a large nonzero constant.

This case is also easy. We note that FFF is a holomorphic function in all the variables. Let's consider the variable qkj=zk−wjq_{kj}=z_k-w_jqkj​=zk​−wj​. Given this is a linear transformation the function had better be a holomorphic function of qkjq_{kj}qkj​ too. Hence the FCC is not satisfied, and these states cannot be singlets.

In conclusion: the only singlet is the (n+1,n+1,n)(n+1, n+1, n)(n+1,n+1,n) state.

So what's the mysterious (112)(112)(112) state?

As it turns out this is really a misnomer. I think it's better thought of a (1ˉ1ˉ2)(\bar{1}\bar{1}2)(1ˉ1ˉ2) state – at filling 2/32/32/3, one can check that filling two hole Landau levels (which can be spin up plus spin down LLLs) of the two flux composite fermion gives the correct filling. This state should be what was found in numerics.

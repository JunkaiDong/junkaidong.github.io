Functional Determinants

These are objects that appear in your path integrals. For example, the simplest path integrals are the path integrals that appear in 0+10+10+1 dimensional quantum mechanics:

Z=∫Dxe−S[x] \mathcal{Z}=\int \mathcal{D}x e^{-S[x]} Z=∫Dxe−S[x]

where one imposes the conditions that x(0)=x(β)x(0)=x(\beta)x(0)=x(β). If the action is quadratic in xxx, then we may diagonalize the kernel of the Gaussian and perform the Gaussian integration. Particularly, if

S[x]=∫0βdτ x(τ)(∂τ2+V(τ))x(τ),S[x]=\int_0^\beta d\tau \ x(\tau) (\partial_{\tau}^2+V(\tau))x(\tau),S[x]=∫0β​dτ x(τ)(∂τ2​+V(τ))x(τ),

then we can write down the partition function as

Z=det⁡(∂τ2+V(τ))−1/2.\mathcal{Z} = \det(\partial_{\tau}^2+V(\tau))^{-1/2}.Z=det(∂τ2​+V(τ))−1/2.

The same thing happens in quantum field theories. If we have a complex bosonic field ϕ\phiϕ, then its coherent state path integral gives

Z=∫Dϕe−∫dτϕˉ(∂τ+H)ϕ=det⁡(∂τ+H)−1,\mathcal{Z} = \int \mathcal{D}\phi e^{-\int d\tau \bar{\phi}(\partial_\tau+\mathcal{H})\phi} = \det(\partial_\tau+\mathcal{H})^{-1},Z=∫Dϕe−∫dτϕˉ​(∂τ​+H)ϕ=det(∂τ​+H)−1,

whereas the fermionic path integral of a fermionic field ψ\psiψ gives

Z=∫Dψe−∫dτψˉ(∂τ+H)ψ=det⁡(∂τ+H).\mathcal{Z} = \int \mathcal{D}\psi e^{-\int d\tau \bar{\psi}(\partial_\tau+\mathcal{H})\psi} = \det(\partial_\tau+\mathcal{H}).Z=∫Dψe−∫dτψˉ​(∂τ​+H)ψ=det(∂τ​+H).

We have to recall that the boundary conditions are important for the path integrals: for the bosonic case, one should use the periodic boundary conditions ϕ(0)=ϕ(β)\phi(0)=\phi(\beta)ϕ(0)=ϕ(β), whereas for the fermionic case one should use anti-periodic boundary conditions ψ(0)=−ψ(β)\psi(0)=-\psi(\beta)ψ(0)=−ψ(β). Using periodic boundary conditions for fermions corresponding to inserting (−1)F(-1)^F(−1)F in the path integral, and thus we would be computing the Witten index.

Now let's recall how determinants are defined. Consider an ordinary matrix – or linear operator M\mathcal{M}M. Let's say it is fully diagonalizable (after all they are dense – apply a perturbation and break all the degeneracies so you don't get Jordan blocks), by which we mean

Mϕn=λnϕn,\mathcal{M}\phi_n = \lambda_n \phi_n,Mϕn​=λn​ϕn​,

and furthermore let us assume that the spectrum is bounded from below, 0<λ1<λ2<…0<\lambda_1<\lambda_2<\dots0<λ1​<λ2​<… The determinant can be written as the product of all the eigenvalues:

det⁡M=∏nλn.\det \mathcal{M} = \prod_n \lambda_n.detM=n∏​λn​.

One should immediately be aware that this is not a good thing to do for general linear operators – after all, the infinite product that we would be computing should be infinitely large. But that is usually not a problem for mathematicians or physicists.

Let us now define an auxiliary wavefunction called the ζ\zetaζ function, corresponding to the operator M\mathcal{M}M. Following the convention in [1], I will suppress the M\mathcal{M}M dependence on ζ\zetaζ, but it's good to keep in mind.

The ζ\zetaζ function is defined as

ζ(s)≡Tr(M−s)=∑n1λns.\zeta(s)\equiv \mathrm{Tr}(\mathcal{M}^{-s})=\sum_{n}\frac{1}{\lambda_n^s}.ζ(s)≡Tr(M−s)=n∑​λns​1​.

When the eigenvalues are positive integers, you get the usual Riemann ζ\zetaζ function. For the Landau level, you get the Hurwitz ζ\zetaζ function with shift 1/21/21/2 – λn=n+1/2\lambda_n=n+1/2λn​=n+1/2. It would be interesting to see if there are natural ways to build quantum mechanical Hamiltonians for general Dirichlet LLL functions, but that's for another day probably.

One can compute the derivative of the ζ\zetaζ function at s=0s=0s=0:

ζ′(0)=−∑nlog⁡λn.\zeta'(0)=-\sum_n\log \lambda_n.ζ′(0)=−n∑​logλn​.

Thus, the functional determinant of M\mathcal{M}M can be related to the derivative:

det⁡M=exp⁡(−ζ′(0)).\det \mathcal{M} = \exp(-\zeta'(0)).detM=exp(−ζ′(0)).

It would have been fantastic if the ζ\zetaζ function is well defined at s=0s=0s=0. In fact, one can see that for the operator M=−∇2\mathcal{M}=-\nabla^2M=−∇2 in ddd dimensions,

ζ(s)∼∫∣k∣>1ddkk2s,\zeta(s) \sim \int_{|k|>1} \frac{d^d k}{k^{2s}},ζ(s)∼∫∣k∣>1​k2sddk​,

which is not convergent when s<d/2s\left.<\right. d/2s<d/2 in general!

To compute the ζ\zetaζ function (and the determinant), we need to introduce proper regularizations. Particularly we introduce the trace of the heat kernel:

K(t)=Tre−Mt=∑ne−λntK(t)=\mathrm{Tr} e^{-\mathcal{M}t}=\sum_{n}e^{-\lambda_n t}K(t)=Tre−Mt=n∑​e−λn​t

which should be generally (more) convergent given that the spectrum {λn}\{\lambda_n\}{λn​} is bounded below. Then, one may perform the Mellin transform to get the zeta function:

ζ(s)=1Γ(s)∫0∞dt ts−1K(t).\zeta(s) = \frac{1}{\Gamma(s)}\int_0^\infty dt \ \ t^{s-1} K(t).ζ(s)=Γ(s)1​∫0∞​dt ts−1K(t).

(We note that the original Riemann ζ\zetaζ function is analytically continued onto the complex plane using exactly this formula, with K(t)=∑e−nt=(et−1)−1K(t) = \sum e^{-nt}=(e^{t}-1)^{-1}K(t)=∑e−nt=(et−1)−1, and the Epstein ζ\zetaζ functions with K(t)K(t)K(t) being some sort of Jacobi θ\thetaθ functions. Indeed, if one considers the Mellin transform of the θ\thetaθ function, one may relate ζ(s)\zeta(s)ζ(s) to ζ(1−s)\zeta(1-s)ζ(1−s) – but that's also for another day.)

We have now handwavingly defined the functional determinant of a general linear operator M\mathcal{M}M. On to the calculations!

The Gel'fand-Yaglom Trick

A Magical function F\mathcal{F}F

Let's say that you have some sort of magical oracle, who provides you the following function F\mathcal{F}F. F(λ)\mathcal{F}(\lambda)F(λ) exactly has the eigenvalues of M\mathcal{M}M as its simple zeros: F(λ)=0\mathcal{F}(\lambda)=0F(λ)=0 if and only if λ\lambdaλ is in the spectrum of M\mathcal{M}M, and F′(λn)≠0\mathcal{F}'(\lambda_n)\neq 0F′(λn​)=0. One may say that they can easily come up with such a function: just take F(λ)=det⁡(M−λ)\mathcal{F}(\lambda) = \det(\mathcal{M}-\lambda)F(λ)=det(M−λ). Indeed, that is probably right, but this has simply translated the calculation of one functional determinant into another one – which is strictly harder, because F(0)=det⁡(M)\mathcal{F}(0) = \det(\mathcal{M})F(0)=det(M) in this case! Indeed, we will soon prove that det⁡M∝F(0)\det \mathcal{M}\propto \mathcal{F}(0)detM∝F(0), and we will provide another shortcut to calculate F(0)\mathcal{F}(0)F(0).

but first let's see what you can do with a general function F\mathcal{F}F with this property. Let us consider its logarithmic derivative:

dlog⁡F(λ)dλ=F′(λ)F(λ).\frac{d\log \mathcal{F}(\lambda)}{d\lambda} = \frac{\mathcal{F}'(\lambda)}{\mathcal{F}(\lambda)}.dλdlogF(λ)​=F(λ)F′(λ)​.

The logarithmic derivative of F\mathcal{F}F has simple poles at λn\lambda_nλn​ with residue 111, irrespective of what F\mathcal{F}F is. This can be seen as the trace of the resolvent of M\mathcal{M}M:

dlog⁡F(λ)dλ=TrR(λ;M)=Tr1λ−M.\frac{d\log \mathcal{F}(\lambda)}{d\lambda} =\mathrm{Tr} R(\lambda;\mathcal{M})=\mathrm{Tr} \frac{1}{\lambda - \mathcal{M}}.dλdlogF(λ)​=TrR(λ;M)=Trλ−M1​.

Thus, the zeta function can be written as an integral with the following contour:

ζ(s)=12πi∫dλ1λsTr1λ−M=12πi∫dλ1λsdlog⁡F(λ)dλ\zeta(s) =\frac{1}{2\pi i}\int d\lambda \frac{1}{\lambda^s} \mathrm{Tr} \frac{1}{\lambda - \mathcal{M}} = \frac{1}{2\pi i}\int d\lambda \frac{1}{\lambda^s}\frac{d\log \mathcal{F}(\lambda)}{d\lambda} ζ(s)=2πi1​∫dλλs1​Trλ−M1​=2πi1​∫dλλs1​dλdlogF(λ)​

where the blue crosses represent the poles of the resolvent and the black wavy line represent the branch cut of the function 1/λs1/\lambda^s1/λs. As one expects, we now deform the contour of the integral:

Thus the integral becomes

ζ(s)=∫0−∞dλ(1(λ−iϵ)s−1(λ+iϵ)s)dlog⁡F(λ)dλ=sin⁡πsπ∫0−∞dλ1∣λ∣sdlog⁡F(λ)dλ\zeta(s) =\int_0^{-\infty} d\lambda \left(\frac{1}{(\lambda-i\epsilon)^s}-\frac{1}{(\lambda+i\epsilon)^s}\right)\frac{d\log \mathcal{F}(\lambda)}{d\lambda} = \frac{\sin \pi s}{\pi}\int_0^{-\infty} d\lambda \frac{1}{|\lambda|^s}\frac{d\log \mathcal{F}(\lambda)}{d\lambda} ζ(s)=∫0−∞​dλ((λ−iϵ)s1​−(λ+iϵ)s1​)dλdlogF(λ)​=πsinπs​∫0−∞​dλ∣λ∣s1​dλdlogF(λ)​

Thus we may simplify the definition of ζ′(0)\zeta'(0)ζ′(0):

ζ′(0)=−log⁡F(0)+log⁡F(−∞)\zeta'(0) = -\log\mathcal{F}(0)+\log\mathcal{F}(-\infty)ζ′(0)=−logF(0)+logF(−∞)

The functional determinant can thus be expressed as

det⁡M=F(0)F(−∞).\det\mathcal{M} = \frac{\mathcal{F}(0)}{\mathcal{F}(-\infty)}.detM=F(−∞)F(0)​.

Indeed, in practice what one would usually compute is a ratio of functional determinants, which usually converges to something finite:

det⁡Mdet⁡Mfree=F(0)Ffree(0).\frac{\det\mathcal{M}}{\det\mathcal{M}_{\textrm{free}}} = \frac{\mathcal{F}(0)}{\mathcal{F}_{\textrm{free}}(0)}.detMfree​detM​=Ffree​(0)F(0)​.

We have cancelled out the infinite contributions. In fact, one can probably think of this precisely as the regulator for calculating this functional determinants.

…\dots… but where can I buy the magical function?

So in the last section we have seen that you want a magical F(λ)\mathcal{F}(\lambda)F(λ) with simple zeros precisely at the spectrum of the operator M\mathcal{M}M. Let us now construct one.

In this section, we will consider the limited case for Dirichlet boundary conditions: ϕ(0)=ϕ(β)=0\phi(0)=\phi(\beta)=0ϕ(0)=ϕ(β)=0, and quadratic derivative operators:

M=−∂τ2+V(τ).\mathcal{M}=-\partial_\tau^2+V(\tau).M=−∂τ2​+V(τ).

Let's consider the solution of the following differential equation:

Muλ(τ)=λuλ(τ),\mathcal{M}u_{\lambda}(\tau) = \lambda u_{\lambda}(\tau),Muλ​(τ)=λuλ​(τ),

where uλ(0)=0,uλ′(0)=1u_{\lambda}(0)=0, u'_{\lambda}(0)=1uλ​(0)=0,uλ′​(0)=1. In general, since this is just an ordinary second order differential equation with correctly specified boundary conditions. The boundary conditions on u′(0)u'(0)u′(0) is simply a normalization – it will not be important. Here comes the magic.

Consider the quantity uλ(β)u_\lambda(\beta)uλ​(β). I claim that this function is zero precisely for λ\lambdaλ that lives in the spectrum of M\mathcal{M}M with the boundary conditions ϕ(0)=ϕ(β)=0\phi(0)=\phi(\beta)=0ϕ(0)=ϕ(β)=0!

One side is easy to see: if uλ(β)=0u_{\lambda}(\beta)=0uλ​(β)=0, then it becomes an eigenstate with the correct boundary condition, and thus λ\lambdaλ is in the spectrum.

For the other side, consider the Wronskian W(ϕn,uλ)=ϕn′uλ−uλ′ϕnW(\phi_n,u_{\lambda}) = \phi_n'u_{\lambda}-u_{\lambda}'\phi_nW(ϕn​,uλ​)=ϕn′​uλ​−uλ′​ϕn​. Now, W′(τ)=ϕn′′uλ−uλ′′ϕn=(λ−λn)ϕnuλW'(\tau) = \phi_n''u_{\lambda}-u_{\lambda}''\phi_n=(\lambda-\lambda_n)\phi_n u_\lambdaW′(τ)=ϕn′′​uλ​−uλ′′​ϕn​=(λ−λn​)ϕn​uλ​. Now we integrate and find out:

uλ(β)=W(β)−W(0)ϕn′(β)=λ−λnϕn′(β)∫0βdτϕnuλu_{\lambda}(\beta) = \frac{W(\beta)-W(0)}{\phi_n'(\beta)} = \frac{\lambda-\lambda_n}{\phi_n'(\beta)}\int_0^\beta d\tau \phi_n u_\lambdauλ​(β)=ϕn′​(β)W(β)−W(0)​=ϕn′​(β)λ−λn​​∫0β​dτϕn​uλ​

and thus, if λ=λn\lambda=\lambda_nλ=λn​, uλ(β)=0u_{\lambda}(\beta)=0uλ​(β)=0.

We have thus proven that uλ(β)u_{\lambda}(\beta)uλ​(β) is our magic function:

F(λ)=uλ(β)!\mathcal{F}(\lambda)=u_{\lambda}(\beta)!F(λ)=uλ​(β)!

From the discussion above, we know that F(λ)\mathcal{F}(\lambda)F(λ) is almost det⁡(M−λ)\det(\mathcal{M}-\lambda)det(M−λ). This equivalence can be most easily understood as a Liouville argument: uλ(β)u_{\lambda}(\beta)uλ​(β) has the same zeros as F(λ)\mathcal{F}(\lambda)F(λ) and det⁡(M−λ)\det(\mathcal{M}-\lambda)det(M−λ), and all of them are analytic in λ\lambdaλ, which means that they must be the same up to a (possibly infinite) constant – which gets cancelled when one calculates the ratio of determinants.

More boundary conditions

We have now obtained the functional determinant for Dirichlet boundary conditions. But the boundary conditions that we really care about are periodic or antiperiodic boundary conditions. Fortunately one can still perform the same procedure. Let us write down the general form of boundary conditions as

M(ϕ(0)ϕ′(0))+N(ϕ(β)ϕ′(β))=0M \left(\begin{array}{c}\phi(0)\\ \phi'(0)\end{array}\right) + N \left(\begin{array}{c}\phi(\beta)\\ \phi'(\beta)\end{array}\right)=0M(ϕ(0)ϕ′(0)​)+N(ϕ(β)ϕ′(β)​)=0

For Dirichlet,

M=(1000),N=(0010).M = \left(\begin{array}{cc}1 & 0\\ 0 & 0\end{array}\right), N=\left(\begin{array}{cc}0 & 0\\ 1 & 0\end{array}\right).M=(10​00​),N=(01​00​).

For Neumann,

M=(0001),N=(0100).M = \left(\begin{array}{cc}0& 0\\ 0 & 1\end{array}\right), N=\left(\begin{array}{cc}0& 1\\ 0 & 0\end{array}\right).M=(00​01​),N=(00​10​).

For periodic,

M=(1001),N=−(1001).M = \left(\begin{array}{cc}1& 0\\ 0 & 1\end{array}\right), N=-\left(\begin{array}{cc}1& 0\\ 0 & 1\end{array}\right).M=(10​01​),N=−(10​01​).

For antiperiodic,

M=(1001),N=(1001).M = \left(\begin{array}{cc}1& 0\\ 0 & 1\end{array}\right), N=\left(\begin{array}{cc}1& 0\\ 0 & 1\end{array}\right).M=(10​01​),N=(10​01​).

One can then construct two solutions: uλ1(0)=1,uλ1′(0)=0u_{\lambda 1}(0)=1,u'_{\lambda 1}(0)=0uλ1​(0)=1,uλ1′​(0)=0, and uλ2(0)=0,uλ2′(0)=1u_{\lambda 2}(0)=0,u'_{\lambda 2}(0)=1uλ2​(0)=0,uλ2′​(0)=1. If any linear combinations of the two solutions satisfy the boundary conditions, then λ\lambdaλ is in the spectrum. We simply need to express the condition that there exist a linear combination as an algebraic equation.

Indeed, one can check that

F(λ)=det⁡(M+N(u1λ(β)u2λ(β)u1λ′(β)u2λ′(β)))\mathcal{F}(\lambda)=\det\left(M+N\left(\begin{array}{cc}u_{1\lambda}(\beta)& u_{2\lambda}(\beta)\\ u_{1\lambda}'(\beta) & u_{2\lambda}'(\beta)\end{array}\right)\right)F(λ)=det(M+N(u1λ​(β)u1λ′​(β)​u2λ​(β)u2λ′​(β)​))

correctly hits zero as the boundary condition is met (and thus we have λ=λn\lambda=\lambda_nλ=λn​ for some nnn). The functional determinant is therefore

det⁡(−∂τ2+V(τ))=F(0)=det⁡(M+N(u1,λ=0(β)u2,λ=0(β)u1,λ=0′(β)u2,λ=0′(β))).\det\left(-\partial_\tau^2+V(\tau)\right)=\mathcal{F}(0)=\det\left(M+N\left(\begin{array}{cc}u_{1,\lambda=0}(\beta)& u_{2,\lambda=0}(\beta)\\ u_{1,\lambda=0}'(\beta) & u_{2,\lambda=0}'(\beta)\end{array}\right)\right).det(−∂τ2​+V(τ))=F(0)=det(M+N(u1,λ=0​(β)u1,λ=0′​(β)​u2,λ=0​(β)u2,λ=0′​(β)​)).

Examples: bosons and fermions

In this section we will compute functional determinants for a free boson and a free fermion, both at frequency ω\omegaω. The boson partition function is

Zb=det⁡(∂τ+ω)P−1det⁡(∂τ)P−1,\mathcal{Z}_b =\frac{\det(\partial_\tau+\omega)^{-1}_{P}}{\det(\partial_\tau)^{-1}_{P}}, Zb​=det(∂τ​)P−1​det(∂τ​+ω)P−1​​,

whereas the fermionic partition function is

Zf=det⁡(∂τ+ω)APdet⁡(∂τ)AP.\mathcal{Z}_f = \frac{\det(\partial_\tau+\omega)_{AP}}{\det(\partial_\tau)_{AP}} .Zf​=det(∂τ​)AP​det(∂τ​+ω)AP​​.

One might think that these are single derivative operators and it would not be possible to calculate the partition functions. This is not true. In fact one needs to apply a time reversal to the determinant, which leaves the boundary conditions invariant. This flips the sign of ∂τ\partial_\tau∂τ​. Multiplying them together and taking the square root, we get

Zb=det⁡(−∂τ2+ω2)P−1/2det⁡(−∂τ2)P−1/2,Zf=det⁡(−∂τ2+ω2)AP1/2det⁡(−∂τ2)AP1/2.\mathcal{Z}_b = \frac{\det(-\partial_\tau^2+\omega^2)^{-1/2}_{P}}{\det(-\partial_\tau^2)^{-1/2}_{P}} , \quad \mathcal{Z}_f =\frac{\det(-\partial_\tau^2+\omega^2)^{1/2}_{AP}}{\det(-\partial_\tau^2)^{1/2}_{AP}} .Zb​=det(−∂τ2​)P−1/2​det(−∂τ2​+ω2)P−1/2​​,Zf​=det(−∂τ2​)AP1/2​det(−∂τ2​+ω2)AP1/2​​.

Let us use the tricks learned in the last section to calculate the determinants. The two solutions we have are

u1,λ=0(τ)=cosh⁡ωτ,u2,λ=0(τ)=sinh⁡ωτω.u_{1,\lambda=0}(\tau)=\cosh\omega\tau, \quad u_{2,\lambda=0}(\tau) = \frac{\sinh\omega\tau}{\omega}.u1,λ=0​(τ)=coshωτ,u2,λ=0​(τ)=ωsinhωτ​.

Plugging them into the formulas above, we get

det⁡(−∂τ2+ω2)P=−sinh⁡2βω2,det⁡(−∂τ2+ω2)AP=cosh⁡2βω2.\det(-\partial_\tau^2+\omega^2)_P = -\sinh^2\frac{\beta\omega}{2},\quad \det(-\partial_\tau^2+\omega^2)_{AP}=\cosh^2\frac{\beta\omega}{2}. det(−∂τ2​+ω2)P​=−sinh22βω​,det(−∂τ2​+ω2)AP​=cosh22βω​.

One might be confused why I am getting something that is negative when I'm computing a positive product: the honest answer is I don't know. But very nicely, the negative sign drops out when we consider the ratio of determinants. (Also we don't consider anomalies here so I can ignore phases of the partition function.)

Zb=βω2sinh⁡βω2,Zf=cosh⁡βω2.\mathcal{Z}_b =\frac{\beta\omega}{2\sinh \frac{\beta\omega}{2}},\quad \mathcal{Z}_f = \cosh\frac{\beta\omega}{2}. Zb​=2sinh2βω​βω​,Zf​=cosh2βω​.

which is exactly what you expect. More examples can be found in [1], but that's beyond the scope of this post.

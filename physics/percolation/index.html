Model

The model is introduced by two mathematical giants, Paul Erdős and Alfréd Rényi in 1960. It has the following inputs:

  1. a set of nnn points as vertices.

  2. a probability p=dn−1p=\frac{d}{n-1}p=n−1d​ which describes the probability of having an edge between any pair of vertices.

Note that the model lives in 000 dimensions: there is no embedding into a manifold. We are interested in the limit where n→∞n\to\inftyn→∞ and ddd fixed. The probability is defined such that ⟨degree of any given vertex⟩=d\langle \textrm{degree of any given vertex} \rangle=d⟨degree of any given vertex⟩=d. Macroscopically, this model describes a set of graphs.

Through this calculation, we find that forming cycles are costly: the only subgraphs whose quantities scale linearly with nnn are trees.

All the graphs we studied before are finite, small clusters compared to the system size nnn: there might be the case where there exists a cluster whose size scales linearly with nnn. When this happens, we say that the system percolates. The central question to ask in this note is: when will the system percolate?

At this moment, the readers informed in physics should feel the plug in their stomach: this vaguely has the flavor of asking when a bosonic system will condense. This will be made more precise in a later section. The physical approach we often take on BECs is that, we calculate the thermal distribution of bosonic particles and see if the thermal distribution can take all the particles; if it can't take all the particles, the amount that overflows (which is a macroscopic quantity) condenses into the ground state. We will ask a similar question here: What fraction of vertices are in finite clusters as n→∞n\to\inftyn→∞?

BEC as a graph model

I would like to sketch how BEC becomes a graph model. Consider the partition function of NNN bosons:

Forgetting all the macroscopic details, we can try to study the structure of the permutation ppp. Which kinds of permutations dominate the sum? It is not unimaginable that when BEC condensation happens, the sum is dominated by permutations containing a cycle of length order nnn - this is called cycle percolation, and it has been studied intensively by this paper and this paper.
